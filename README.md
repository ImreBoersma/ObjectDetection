
# Getting started

1. Download the model from [ONNX Model zoo RetinaNET](https://github.com/onnx/models/tree/master/vision/object_detection_segmentation/retinanet).
2. Place the model in `ObjectDetection/assets/Model`.
3. Make sure the model name is correct in [Program.cs](/ObjectDetection/Program.cs#L26) on line 26
